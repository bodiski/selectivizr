Selectivizr
-----------

**CSS3 for IE 6-9**

## Selectivizr ##

Selectivizr是一个为IE6-8提供css3选择器的js库，他的官方网站是：http://selectivizr.com/

## Fork ##

这个fork整合了PIE修改版的PIE，从而让IE6-9获得css3的选择器之外，或获得了圆角、阴影、渐变色资源等css3属性的支持

使用方式：
```HTML
<!--[if lte IE 9]>
	<script type="text/javascript" src="selectivizr.js"></script>
<![endif]-->
```
